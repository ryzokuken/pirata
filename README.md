# Pirata

An open-source, community-focused pirate RPG set in the golden age of piracy.
Exploration, factions, reputation, stealth, and crime — combat is what happens
when your plan fails.

- **Design:** [docs/superpowers/specs/2026-07-05-pirata-design.md](docs/superpowers/specs/2026-07-05-pirata-design.md)
- **License:** GPLv3 (code); assets individually credited in [ATTRIBUTION.md](ATTRIBUTION.md)

## Develop

    pnpm install
    pnpm dev

The dev server hot-reloads on changes to client, core, and content files.
